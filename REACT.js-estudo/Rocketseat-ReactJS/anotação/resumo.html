Bundlers & Compilers

Compilers são compiladores de código, ou seja, ferramentas que convertem nosso código de um formato para o outro. Exemplo quero executar meu código React dentro do navegador e este navegador não suporta algumas funcionalidades do JS, teria que pegar esse código JS mais atualizado e passar para um que o navegador entenda.
O compiler mais conhecido é o Babel.

Um bundle conhecido é o webpack que suporta a gente importar arquivos de diferentes formatos. exmeplo quando um web page tem em sua composição arquivos sass. .cjs .hbs .jpg .png, o bunde converte para os mais basicos que são .js .css .png.

Tem o VITE que usa por padrão o ESM e não é necessário Bundle. O VITE faz o processo de compiling de forma automatica sem precisar do babel. O vite ja tras algumas coisas configuradas pra gente, exmeplo o fast refresh(alterar nossa aplicações no Browser de forma instantanea)
